ポケモン捕獲率計算ver0.01

■概要
捕まえたいポケモンの名前とレベルを入力するだけで捕獲率を計算できます。

BDSP全国図鑑（No.001-フシギバナからNo.493-アルセウス）の範囲のみ対応。

※サファリモード未実装
※捕獲クリティカル未実装
※揺れ判定未実装

未実装要素が多いため、表示確率より実際には捕まえやすいor捕まえにくいという事は起こり得ます。
予めご了承頂けると幸いです。

■使い方
1.[ポケモン名]テキストボックスに捕まえたいポケモンの名前を入力します
2.[レベル]テキストボックスに捕まえたいポケモンのレベルを入力します

3.上記入力が適切であるとき、画面左下の[捕獲率を計算する]ボタンが有効になります
　押すと捕獲率を計算し、画面左上に表示します。

4.状態異常やHPを1にする、使用ボールを変えた居場合は各選択肢のチェックを変えたうえで
　[捕獲率を計算する]ボタンを押すことで、対応する捕獲率を計算できます。



■アプリ画面の要素の詳細説明

□[ポケモン名]　テキストボックス。日本語版のポケモン名のみ有効です。No.493までのポケモンに対応しています。
	具体的には同梱のPokemonDate.csvのB列目(2列目)と一致する場合有効となります。
	PokemonDate.csvにはアルセウスまでのデータしか入っていません。
　
□[レベル]　テキストボックス。半角数字0~100のみ有効です。
	このアプリの内部ではポケモン名とレベルを渡すことで、そのポケモンのHP種族値を参照し
	HP実数値を計算しています。
	実は捕獲率計算に「HPの数値はほんのちょっとしか影響しない」ので、
	正確なレベルを入力しなくてもよく、適当に50とか入れておけば大丈夫です。

□[残りHP] 　
　○[まんたん] HPを減らさない場合、こちらを選びます。起動時にデフォルトで選択されています。
　○[のこり1]　減らせる限界のHP1状態を基準に捕獲率を計算します。
	※HP満タン未満HP2以上の選択肢は未実装

□[状態異常]
　○[なにもしない] 相手のポケモンに何も状態異常を付与しない場合、こちらを選びます。起動時にデフォルトで選択。
　○[まひ] まひ、やけど、どく状態を付与する場合。
　○[ねむり] ねむり、こおり状態を付与する場合。
	
□[使うボール]　-クイック以降は最適条件を選んだ時の最大値を計算に反映させます。
	それぞれの選択肢は固定の補正値を保持しているだけなので、
	選択肢にないボールと同じ補正値をもつものを選ぶことで若干代用できることがあります。

　○[モンスターボール] =1 [条件を満たさない特殊ボール],[フレンドボール][ゴージャスボール][プレミアボール][ヒールボール]
　○[スーパーボール]=1.5 [コンペボール],[サファリボール]
	※ただしサファリは捕獲計算式が異なるため正確な計算結果にはなりません
　○[ハイパーボール]=2 [レベルボール(先頭の手持ちポケモンLV1/2≦ 捕まえるポケモン<LV1/1)]
　○[クイックボール]=5 [クイックボール(1T)][ウルトラボール(UB)][ルアーボール(釣り)]
　○[ダークボール]=3 [ダークボール(夜,洞窟)]
　○[リピートボール]=3[リピートボール(捕獲済)]
　○[タイマーボール]=4 [タイマーボール(10T~)][ドリームボール(眠り)][スピードボール(S種族値100以上)]etc

	※未実装補正値例
	0.1[ウルトラボール(UB以外)]
	1.3~3.7[タイマーボール(1T目1.3,9T目まで0.3ずつ増えていく)]
	3.5 [ダイブボール][ネットボール]
	8 [レベルボール(LV1/4未満)][ラブラブボール]

□[捕獲率を計算する]　ボタン。ポケモン名とレベルの入力が適切であれば有効になります。
	このボタンを押すことで計算式に数値が渡され、捕獲確率を計算できます。
	推定捕獲率は(HP個体値0の時の確率)%～(HP個体値31の時の確率)%で表示されます。
	起動直後は0.0000%です。
	100%以上ならば捕獲確定です。実際には99%ぐらいでも確定かもしれません（未検証）
